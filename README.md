# Paint

A simple Apple-style Paint web app built with plain HTML, CSS, and JavaScript.

## Features

- Pencil
- Eraser
- Color picker
- Rectangle and ellipse tools
- Brush size slider
- Color swatches
- Undo and redo
- Clear canvas
- PNG save

## Run locally

Open `index.html` in a browser, or serve the folder locally and visit the local URL.
